#UTS Praktikum DPW
Pada tugas ini, dilakuka _redesain_ pada halaman utama web Fakultas Teknik Universitas Riau, dengan tujuan untuk meningkatkan tampilan visual, kenyamanan pengguna, serta memastikan web tersebut dapat berjalan dengan baik di berbagai perangkat (desktop dan mobile).
Untuk perubahan yang kami lakukan tidak terlalu banyak dan signifikan. Perubahan paling banyak kami lakukan untuk tampilan mobile. Berikut beberapa perubahan yang kami lakukan.
1. Perubahan Warna Background Header (Mobile)
   Pada bagian menu atas (header), kami telah mengubah background berwarna putih. Karena sebelumnya, untuk tampilan di mobile, warna background header sama dengan warna teks navigasi (putih),         sehingga sangat mengganggu kenyamanan user.
2. Perubahan Ukuran Gambar Slider (Mobile)
   Perubahan selanjutnya yaitu ukuran gambar slider pada mobile. Di halaman utama web FT UNRI, terdapat slider yang menampilkan beberapa gambar, namun, pada tampilan mobile, gambar tersebut       	memiliki ukuran yang kurang pas, gambar yang tampil terlihat terpotong di bagian kiri dan kanan sehingga tidak tampil sempurna, sehingga kami membuat perubahan yang membuat ukuran slider gambar 	menjadi terlihat sempurna tanpa terpotong.
3. Perubahan Pada Bagian Berita (Kabar FT UNRI)
   Pada tampilan berita (Kabar FT UNRI) pada web ini terlihat terlalu polos dengan teks yang memiliki ukuran cukup besar, sehongga kami lakukan perubahan style di sini. Sebelumnya hanya ada teks 	"Kabar FT UNRI", kami ubah seperti ala ala blog dengan kalimat "Berita Terbaru" di atas sebelah kiri berita dengan sedikit aksen berwarna merah.
4. Perubahan UX Ketika Menu Navbar Muncul
   Pada web aslinya, ketika menu navbar muncul (klik garis 3), maka user tidak bisa scroll sebelum menutup atau memilih menu. Di sini kami ubah agar user tetap bisa scroll meski menu navbar masih 	muncul. Hal ini bertujuan agar user tidak bingung saja.
5. Perubahan Jumlah Berita yang Terlihat
   Sebelum di-_redesain_, jumlah berita yang muncul ada  6, setelah kami desain ulang, berita yang muncul kami batasi 3, dan jika pengguna ingin melihat berita lain selengkapnya, maka harus klik 	"Berita Lainnya". Hal ini bertujuan agar pengguna mobile tidak perlu scroll terlalu ke bawah untuk menemukan navigasi "Berita Lainnya".

#Kesimpulan
Dengan dilakukannya perubahan-perubahan tersebut, website menjadi lebih responsif, memiliki tampilan menarik, dan user friendly, sehingga pengalaman mengakses web FT UNRI menjadi lebih baik.
